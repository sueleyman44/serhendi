# 📄 README – Serhendi Kulturverein Berlin (Webseite)

## 🌐 Projektübersicht
Dieses Projekt enthält eine einfache, statische Webseite für den **Serhendi Kulturverein Berlin**.  
Die Seite dient als digitale Visitenkarte des Vereins und bietet grundlegende Informationen zu:

- Startseite  
- Veranstaltungen  
- Schulungen  
- Login‑Popup (Frontend‑Simulation)

Die Seite ist vollständig in **HTML**, **CSS** und **JavaScript** umgesetzt und benötigt keinen Server oder zusätzliche Frameworks.

## ✨ Funktionen & Inhalte

### 🔹 Navigation
Die Navigationsleiste ermöglicht das Scrollen zu den Bereichen:
- Startseite  
- Veranstaltungen  
- Schulungen  
- Login‑Popup

### 🔹 Startseite
- Begrüßungstext  
- Bild (Moschee mit Rosengarten)  
- Buttons zu Veranstaltungen & Schulungen  

### 🔹 Veranstaltungen
Ein Platzhalterbereich, in dem zukünftige Events eingetragen werden können.

### 🔹 Schulungen
Ein Bereich für Bildungsangebote des Vereins.

### 🔹 Login‑Popup
Ein modales Fenster, das per Klick geöffnet wird.  
Es enthält:
- Benutzername‑Feld  
- Passwort‑Feld  
- Anmelde‑Button  

> Hinweis: Das Login ist rein visuell und besitzt keine Backend‑Funktion.
